This is the github repository for Machine Learning project under Zine.
Problem Statement: https://drive.google.com/file/d/1AH6nDERVY4C37SLjrm1tlI0QWOdiOhQl/view

The objective of this project is to design and train a Machine Learning
system capable of detecting DeepFake (spoofed) voice samples and
distinguishing them from real human speech.
DataSet used: https://www.kaggle.com/datasets/awsaf49/asvpoof-2019-dataset (ASVspoof 2019)
